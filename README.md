# Shopify Dawn Bundle Builder

A production-ready Bundle Builder section for Shopify's Dawn theme that allows merchants to create customizable product bundles with live pricing, variant selection, inventory validation, and AJAX cart support.

Designed for **Online Store 2.0** themes and built using **Liquid, Vanilla JavaScript, CSS, and Shopify's AJAX Cart API**.

---

## Features

- Dawn Theme Compatible (v15+)
- Shopify Online Store 2.0
- Merchant-configurable products
- Live bundle pricing
- Percentage-based bundle discount
- Product variant selection
- Inventory validation
- AJAX Add to Cart
- Responsive design
- Accessible markup
- No external libraries
- Lightweight and easy to customize

## Installation

1. Copy the files into your Shopify theme.

2. Add the Bundle Builder section from the Theme Editor.

3. Select bundle products.

4. Set your preferred bundle discount.

5. Save.
