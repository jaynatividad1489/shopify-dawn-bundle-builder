# Testing Checklist

Before publishing the section, verify the following:

* Bundle Builder appears in the Theme Editor.
* Products display correctly.
* Images load properly.
* Variant selectors work.
* Prices update when variants change.
* Subtotal updates correctly.
* Discount calculation is correct.
* Total calculation is correct.
* Add Bundle button adds all products to the cart.
* Success and error messages display correctly.
* Layout works on desktop, tablet, and mobile.
