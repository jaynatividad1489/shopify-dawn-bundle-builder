# Installation Guide

## Requirements

* Shopify Online Store 2.0
* Dawn Theme (recommended)
* Shopify Admin access

## Installation

### Step 1

Copy the following files into your theme:

* `sections/bundle-builder.liquid`
* `snippets/bundle-product-card.liquid`
* `assets/bundle-builder.css`
* `assets/bundle-builder.js`

### Step 2

Save all files.

### Step 3

Go to:

**Online Store → Themes → Customize**

### Step 4

Click **Add section**.

Select **Bundle Builder**.

### Step 5

Add Product blocks.

Choose the products you want to include in the bundle.

### Step 6

Configure:

* Heading
* Description
* Bundle Discount
* Button Label
* Color Scheme

### Step 7

Save the theme.

Your Bundle Builder is now ready to use.
