# Customization

The Bundle Builder section can be customized directly from the Shopify Theme Editor.

## Section Settings

### Heading

Changes the main title displayed above the bundle.

### Description

Displays supporting text below the heading.

### Bundle Discount

Applies a percentage discount to the calculated bundle total.

### Button Label

Changes the Add Bundle button text.

### Color Scheme

Uses the selected Shopify color scheme for the section.

## Product Blocks

Each Product block allows you to select one Shopify product.

You can add up to six products to a bundle.

Products with multiple variants automatically display a variant selector.
